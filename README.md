## :clipboard: Description
findove-api is the api for the [Findove App](https://github.com/ErickLuizA/findove)



## :building_construction: Technologies
- [Nodejs](https://www.nodejs.org)


## :rocket: Getting Started

## 1. Download the repository

```shell
 $ git clone https://github.com/ErickLuizA/findove-api.git
```

## 2. Installation

``` shell
# Go into the repository
$ cd findove-api

# Install dependencies
$ yarn install
```

## 3. Execution

```shell
$ yarn start
```
